加密币

冷却逻辑：再主要国家政策发布前后2小时，不进行任何交易。

波动监控：
    波动周期：统计市场的，再波动幅度大时，立即交易；套现
    波动方向：超过n个周期K线指向一个方向时，立即停止交易并离场，等待波动来临

盈亏比: 1.9 : 1 比传统的2:1稳0.1

仓位管理：
    单笔仓位：不超过总资金的 2-5%
    总仓位上限：同时持仓不超过总资金的 20-30%
    杠杆限制：不使用或限制在 2-3 倍以内
    连续亏损保护：连续 3 次亏损后，降低仓位至 50% 或暂停交易

止损止盈：
    固定止损：单笔交易最大亏损不超过本金的 2%
    移动止损：盈利超过 1% 后，止损线跟随上移（Trailing Stop）
    分批止盈：盈利达到 50% 目标时止盈 50%，剩余部分跟随移动止损
    时间止损：持仓超过 24 小时未达到目标，强制平仓

流动性风险：
    滑点控制：单笔交易滑点不超过 0.5%，超过则取消交易
    最小流动性：只交易 24h 交易量 > 1000 万美元的币种
    大单拆分：单笔交易金额 > 10 万美元时，拆分为多笔执行
    深度检查：下单前检查订单簿深度，确保能完全成交

技术风险：
    API 限流：遵守币安 API 限流规则，实现指数退避重试
    网络异常：API 调用失败超过 3 次，暂停交易并告警
    交易所故障：检测到币安 API 异常或维护，暂停交易并告警
    数据同步：定期校验账户余额、持仓数据，发现不一致立即告警

资金安全：
    热钱包限额：交易所热钱包余额不超过总资金的 10%
    冷钱包存储：90% 资金存储在冷钱包或硬件钱包
    提现风控：单日提现限额，大额提现需要二次确认
    多签保护：冷钱包使用多签（2/3 或 3/5）

市场异常保护：
    闪崩/闪涨检测：价格在 1 分钟内波动超过 10%，暂停交易
    异常交易量：24h 交易量突然放大 5 倍以上，暂停该币种交易
    价格偏离保护：现货价格与多个数据源偏差 > 3%，暂停交易
    交易所异常：检测到交易所出现异常交易或暂停交易，立即平仓

监控与告警：
    实时监控：账户余额、持仓盈亏、API 状态、网络延迟
    异常告警：亏损超过阈值、API 异常、价格异常、余额异常
    日志审计：所有交易决策、执行结果、异常情况完整记录
    每日报告：自动生成每日交易报告（盈亏、交易次数、胜率）

回撤控制：
    最大回撤：总资金回撤超过 10%，暂停交易并重新评估策略
    月度回撤：月度回撤超过 5%，降低仓位或暂停交易
    连续亏损：连续 5 笔亏损，强制暂停交易 24 小时

合规与审计：
    交易记录：完整记录所有交易（时间、价格、数量、手续费）
    税务准备：按国家/地区要求记录交易数据，便于税务申报
    定期审计：每月进行策略回测和资金审计，确保策略有效性
